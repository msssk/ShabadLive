SbdLive: shabad search and display

ShabadLive aims to be the most flexible, beautiful, and easy to use shabad display software. Existing programs like SikhiToTheMax and iSearchGurbani serve the sangat well, but ShabadLive will improve their design by simplifying the job of the person controlling the shabad display and providing unlimited flexibility in adding and configuring displays.

How? Let's start with the search interface

The iGurbani mobile app has a time-tested and proven interface for searching gurbani. ShabadLive will provide a very similar interface. A key feature is that the interface will be usable on mobile devices as well as computers so that the person controlling the display can sit comfortably with the sangat and enjoy the kirtan.

What will it look like?

However you want! Some nice defaults will be included, but anyone with some basic HTML knowledge will be able to fully customize the display, including background, font, and colors. This enables everyone to create the best looking display that integrates well with its surroundings.

What are some interesting features?

  • Optimized text rendering: long lines will be split so the font can be larger and more visible
  • Multiple displays: any number of web browsers can connect and display the shabad
  • Independently configurable displays: each connected display can display a different language
  • Broad language support: Gurmukhi, English, Punjabi, Spanish, Hindi

Questions?

The program will be freely distributed, along with its source code, so anyone can use it for free and even modify it.
